## Description
This back-end application allows users to run the database for e-commerce websites using Express.js API and Sequelize to connect to the MYSQL database. As a user you can create a development database, seed it with test data, and sync Sequelize models to MySQL database. As a user you have the ability to GET, POST, PUT and DELETE routes to display and manipulate data in the userâ€™s database.
